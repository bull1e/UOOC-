# UOOC
适用于UOOC网课平台的刷课脚本

能够静音二倍速自动刷课；可直接挂在后台（前提是不把UOOC视频页面最小化），让你可以全屏专注于其他；遇到测验语音播报提醒，遇到弹窗小测直接秒杀

食用方法：

第一步：下载插件压缩包
在当前 GitHub 页面，找到右上角绿色的 <> Code 按钮并点击。

在弹出的菜单中，点击最下方的 Download ZIP。

下载完成后，务必将这个 ZIP 压缩包解压（提取出一个普通的文件夹，请记住解压到了电脑的哪个位置）。

第二步：打开 Chrome 开发者模式
打开你的 Chrome 浏览器。

在浏览器最上方的地址栏中，复制并粘贴这段地址后回车：chrome://extensions/

在打开的扩展程序页面，找到右上角的 “开发者模式” 开关，将其打开。

第三步：加载插件
开发者模式打开后，页面左上角会出现三个按钮，点击第一个 “加载已解压的扩展程序”。

在弹出的窗口中，找到并选中你刚才解压出来的那个文件夹（注意：要选中里面直接包含 manifest.json 文件的那个层级的文件夹，即“刷课”文件夹），然后点击“选择文件夹”。

安装成功后点入UOOC视频界面启动脚本即可

note：
若遇到脚本无法正常运行，大部分情况点击脚本ui界面的刷新键即可解决

脚本中对弹窗小测自动回答的代码参考了kalila-cc和wybbb1制作的“UOOC assistant”（地址为https://greasyfork.org/zh-CN/scripts/413268-uooc-assistant ），十分感谢
